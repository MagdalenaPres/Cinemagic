# **Cinemagic**
Cinemagic is a cinema mobile app.

## **Features**
- Display movies for chosen date
- Watch trailes
- Look at photos
- Display movie details
- Book a seat
- Display all purchased tickets

## **General informations**
- Language: **Kotlin**
- Database: **Firebase Realtime database**
- IDE: **Android Studio**
